# F35-Flight-Simulator
This repo tracks the development of a custom-built throttle system inspired by the F-35, built for a senior design project. The setup includes a physical throttle, sensors for input tracking, and a custom-designed housing/mount—all connected to a Raspberry Pi. The goal is to make the hardware fully compatible with Microsoft Flight Simulator and Prepar3D.

We’re coding in C++ and using DirectInput for communication between the sensors and the flight sim software. The focus is on solid performance and realistic control, not just ease of development.

## Project Deliverables
- ✈️ F-35-style throttle  
- 🎛️ Sensor system for accurate throttle control  
- 🛠️ Custom mount/base housing  
- 💻 C++ codebase for DirectInput integration with Raspberry Pi  

## Goals
- Realistic flight control experience  
- High performance and reliability  
- Compatibility with Microsoft Flight Simulator and Prepar3D
